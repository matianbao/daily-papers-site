1. Ads in AI Chatbots? An Analysis of How Large Language Models Navigate Conflicts of Interest

  • 作者: Addison J. Wu1∗
  • 机构: 1Princeton University 2University of Washington
  • 链接: arXiv | PDF
  • 来源: 📄 arXiv 关键词检索
  • 核心方法: 系统分析 LLM 在面对广告利益冲突时的行为模式。构建了分类框架：从语言学会话准则和 FTC 广告监管两个维度分析 LLM 的广告植入方式——包括隐性推荐、信息选择性呈现、CoT 中的偏向性推理。测试了 GPT、DeepSeek 等主流模型在广告压力下的行为差异。
  • 对比方法/Baselines: GPT-4o、DeepSeek-V3、Claude 等多模型对比
  • 借鉴意义: 直接关系到推荐系统和广告投放的核心问题：当 LLM 成为用户决策的中介，广告商如何影响 LLM 输出？这对 ad serving 和 computational advertising 的未来形态有深远影响。
  • 锐评: 这篇太对口了。把 LLM 当成新的广告媒介来分析，从语言学和监管双重视角拆解利益冲突。发现 CoT 推理中的偏向性尤其隐蔽——模型在 think 阶段就已经被广告主的 prompt 污染了。唯一遗憾是没有测试 retrieval-augmented 场景下的广告注入，这才是实际部署中最危险的。 🔥

2. Act Wisely: Cultivating Meta-Cognitive Tool Use in Agentic Multimodal Models

  • 作者: Shilin Yan1∗†, Kunyu Shi1, †
  • 机构: 2Huazhong University of Science and Technology, 1Accio Team, Alibaba Group
  • 链接: arXiv | PDF
  • 来源: 📰 HF Daily，⬆️ 18
  • 核心方法: 提出 HDPO（Hierarchical Direct Preference Optimization）训练 agentic multimodal 模型的元认知能力——让模型学会判断何时该调用外部工具、何时该依赖内部知识。用 GRPO 做 RL 微调，在 8 个 benchmark 上超越盲目调用工具的 baseline。
  • 对比方法/Baselines: DeepEyesV2、InternVL2.5、Qwen2.5-VL
  • 借鉴意义: 解决了 agent 系统的一个真实痛点：过度依赖工具调用导致效率低下和幻觉。
  • 锐评: 问题定义精准——'blind tool invocation' 确实是当前 agent 系统的通病。HDPO 的层次化偏好优化思路不错，但 18 个 HF upvotes 说明社区反应平平。可能是因为 benchmark 太多但每个改进幅度不大。实用价值大于学术新意。 👀

3. Awakening the Sleeping Agent: Lean-Specific Agentic Data Reactivates General Tool Use in Goedel Prover

  • 作者: Jui-Hui Chung1
  • 机构: 3Department of Computer Science, University of Illinois at Urbana-Champaign, 1Princeton Language and Intelligence, 2Amazon
  • 链接: arXiv | PDF
  • 来源: 📄 arXiv 关键词检索
  • 核心方法: 研究重度 SFT 后模型丧失 tool use 能力的现象。以 Goedel-Prover-V2 为例——在 180 万条形式数学数据上 SFT 后，模型几乎完全丧失工具调用能力。提出 'Lean-specific agentic data' 方法，用少量精心构造的 agentic 数据恢复 tool use，同时不损害数学推理性能。
  • 对比方法/Baselines: Goedel-Prover-V2、BFCL benchmark
  • 借鉴意义: 揭示了 SFT 的一个重要副作用：domain specialization 会抑制 general tool use。对任何做 agent 微调的团队都是警示。
  • 锐评: 问题很实际——你辛辛苦苦训出一个数学专家，结果发现它不会用工具了。这种 capability trade-off 在 agent 训练中很常见但很少被系统研究。Princeton+Amazon 的联合分析比较扎实。不过 'sleeping agent' 这个标题有点标题党了。 👀

4. Beyond Stochastic Exploration: What Makes Training Data Valuable for Agentic Search

  • 作者: Hao, Chuzhan, Feng, Wenfeng, Jiang, Guochao, Quan, Guofeng, Liu, Guohua, Zhang, Yuewei
  • 机构: Alibaba Cloud Computing, {haochuzhan.hcz, liyou.zyw}@alibaba-inc.com
  • 链接: arXiv | PDF

  • 核心方法: 提出 HiExp（Hierarchical Exploration），分析什么样的训练数据对 agentic search 最有价值。发现当前 RL-based search agent 的随机探索策略效率低下，通过层次化的知识提取和确认机制（HEK + CEM）提升训练数据质量。基于 GRPO 训练。

  • 借鉴意义: 为 agentic search 的训练提供了数据价值的分析框架，对构建更高效的搜索 agent 有指导意义。
